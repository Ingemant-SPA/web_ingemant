# 🗺 Roadmap — Web Corporativa Ingemant SPA

Este documento define las etapas del proyecto, desde la creación inicial hasta las integraciones futuras con sistemas IoT y backend corporativo.

---

## ✅ Fase 1 — Preparación del entorno (COMPLETADO)

- Limpieza del servidor Arch
- Eliminación de Cloudflare Tunnel
- Implementación de Tailscale para acceso remoto estable
- Configuración SSH en Arch y Windows
- Creación del repositorio `web_ingemant`
- Configuración de GitHub + Vercel

---

## ✅ Fase 2 — Base del proyecto (COMPLETADO)

- Crear proyecto React + Vite
- Configurar TailwindCSS
- Crear estructura de carpetas
- Definir carpeta fuente para galeria automatica: public/gallery
- Definir placeholders iniciales para logo y textos base
- Crear componentes base:
  - Navbar
  - Footer
  - Hero
- Implementar sistema de tema con switch claro/oscuro
- Definir tokens de color corporativos (linea azul de confianza)
- Crear rutas principales:
  - Home
  - Servicios
  - Proyectos
  - Nosotros
  - Contacto

---

## 🔜 Fase 3 — Contenido corporativo

- Redacción de textos corporativos
- Integración de imágenes reales de proyectos
- Reemplazo progresivo de placeholders por contenido oficial
- Sección de servicios:
  - Aire comprimido
  - Oxigenación
  - Automatización
  - Mantenimiento industrial
- Galería de proyectos autogenerada desde carpeta del repositorio
- Ajuste fino de paleta visual corporativa en modo claro y oscuro

---

## 🔜 Fase 4 — Funcionalidades

- Formulario de contacto funcional (EmailJS o API)
- Animaciones suaves (Framer Motion)
- Optimización SEO
- Favicon y branding corporativo

---

## 🔜 Fase 5 — Despliegue y validación

- Deploy automático en Vercel
- Revisión con CEO y equipo técnico
- Ajustes visuales y de contenido
- Optimización de rendimiento

---

## 🔮 Fase 6 — Integraciones futuras

- Backend FastAPI para formularios y datos
- Dashboard IoT para monitoreo de equipos
- Integración con sistema de mantenimiento
- API para reportes técnicos
- Autenticación para clientes

---

## 🏁 Fase 7 — Versión 2.0

- Panel administrativo
- Gestión de contenido (CMS interno)
- Integración con base de datos PostgreSQL
- Hosting híbrido (Vercel + backend propio)
